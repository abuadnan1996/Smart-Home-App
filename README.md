# Smart-Home-App
This app is designed to control IoT home appliances. Firebase Real time Database is used as database for this app.
<h4> Features </h4>
<ul>
  <li> 6 Appliance Control Switch</li>
  <li> Configurable Firebase Realtime Database</li>
</ul>
